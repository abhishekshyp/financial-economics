## Submission Instructions

Please read the following instructions carefully before submitting your assignment.

### 1. Upload to the Correct Folder

- Upload your assignment **only** to the folder corresponding to the relevant **weekly tutorial**.
- Assignments uploaded to the wrong folder **will not be graded**.

### 2. File Naming Convention (Mandatory)

Rename your file using the following format before uploading:

```text
RollNumber_FullName.extension
```

**Example**

```text
22617001_AbhishekKashyap.pdf
22617001_AbhishekKashyap.xlsx
22617001_AbhishekKashyap.py
22617001_AbhishekKashyap.ipynb
```

**Any file that does not follow this naming convention will not be accepted for grading.**

### 3. Multiple Files

If your assignment consists of multiple files:

- Create a folder using the **same naming convention**.

Example:

```text
22617001_AbhishekKashyap/
```

Place all required files inside this folder and upload the folder to the appropriate weekly assignment directory.

### 4. Accepted File Types

Depending on the assignment, submissions may include one or more of the following:

- PDF (`.pdf`)
- Excel (`.xlsx`)
- Python (`.py`)
- Jupyter Notebook (`.ipynb`)
- ZIP Archive (`.zip`)
- PowerPoint (`.pptx`)
- HTML/CSS/JavaScript files
- Any other format explicitly mentioned in the assignment instructions

### Important

- ✅ Submit **before the deadline**.
- ✅ Verify that your file or folder has been uploaded successfully.
- ✅ Ensure that your submission is placed in the correct weekly folder.
- ✅ Follow the required naming convention exactly.

**No submissions received after the deadline or through email, WhatsApp, Microsoft Teams, or any other platform will be accepted.**
