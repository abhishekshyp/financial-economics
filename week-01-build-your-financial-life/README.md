# Build Your Financial Life

A single-file, static web app for teaching life-cycle finance. No build step, no backend.

## Deploy on GitHub Pages
1. Create a new repository and add `index.html` to the root.
2. Repo **Settings → Pages → Build and deployment → Source: Deploy from a branch**.
3. Select branch `main` and folder `/ (root)`, then **Save**.
4. Your app is live at `https://<username>.github.io/<repo>/` in ~1 minute.

Charts load Chart.js from a CDN, so students need to be online.
